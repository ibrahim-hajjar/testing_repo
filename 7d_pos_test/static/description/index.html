FEATURES

Credit facility management structured for International, National and Islamic Banking system

Compute Interest based on Fixed and Variable Rates and Repayment Cycles

Auto-creates repayment journal entries for Principal and Interest amounts

Dynamic Reporting Interface

HOW IT WORKS

The Configuration

1- Facility Category: Categorize the facility types like; Infrastructure, Advertising, Equipment, Inventory, etc. button

2- Level of Approvals: Set the Approvers and their Bracket of Approval Amount

The Request

Create the Loan Request with Journal and Approver details and Compute the Installments

The Approval/Rejection

The Approvers will either Approve or Reject the Loan Request

The Receipt

Upon receiving the loan amount, change the status to Receive

The Entries

This auto-creates Journal Entries for the repayments (principal and interest amounts, separately)

The Reporting

The credit facility analysis reporting interface is flexible to view and can be grouped/filtered as desired
