FEATURES

Categorizes your stock locations which makes it easier to view only the specifics, not everything

Cumulates all your Non-Stock and Stock locations using this module

Some examples are; POS and Warehouse locations hold your Sellable Stock while Display, Assets, Furniture, and Fixture locations hold your non-sellable (Non-Stock)

HOW IT WORKS

Create the categories of locations; POS, Warehouse, Consumables, Furniture & Fixtures, Assets, Samples, etc

Configure the internal stock location under the Inventory > Configuration menu; in Debug mode you will find a field [stock_type] where you can map these Categories

While running reports or using filters in the Inventory App; you can define which Stock_Type you require to view its stock information

While configuring your stock dashboard, you can pull data of Stock and Non-Stock separately
